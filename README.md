# Solar Monitoring System Using Arduino

## Overview

This project is an Arduino-based Solar Monitoring System designed to monitor the performance of a solar panel in real time. The system measures voltage, current, power, and energy generation and displays the data on a 16x2 LCD.

## Features

* Real-time voltage measurement
* Real-time current measurement using ACS712 sensor
* Power calculation
* Energy monitoring
* LCD display output
* Arduino-based implementation
* Proteus simulation support

## Components Used

* Arduino Uno
* ACS712 Current Sensor
* Voltage Sensor Module
* 16x2 LCD Display
* Solar Panel
* Connecting Wires
* Power Supply

## Working Principle

The voltage sensor measures the solar panel voltage, while the ACS712 sensor measures current flow. Arduino processes these readings and calculates power and energy values. The results are displayed on the LCD for continuous monitoring.

## Results

The system successfully monitors:

* Voltage (V)
* Current (A)
* Power (W)
* Energy (Wh)

in real time.
